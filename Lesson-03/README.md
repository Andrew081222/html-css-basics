# Lesson 03 — CSS Hover / Active & Transitions

This folder contains my completed exercises from **Lesson 03** of the *SuperSimpleDev HTML & CSS Full Course*.

## What I learned
- Using pseudo-classes:
  - `:hover` to change styles when the mouse is over an element
  - `:active` to change styles while clicking
- Adding smooth effects using `transition`
- Improving button/link feedback using:
  - `opacity`
  - `background-color` and `color` changes
  - `border` changes
  - `box-shadow`

## Exercises included
- **3a — Button opacity on hover** (`3a-button-opacity-hover.html`)
- **3b — Button background-color on hover** (`3b-button-shadow-hover.html`)
- **3c — Button box-shadow on hover** (`3c-button-background-shadow.html`)

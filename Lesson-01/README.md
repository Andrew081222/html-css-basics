# Lesson 01 — HTML Basics (Buttons, Text, Links)

This folder contains my completed exercises from **Lesson 01** of the *SuperSimpleDev HTML & CSS Full Course*.

## What I learned
- Creating basic HTML structure
- Using core HTML elements:
  - `<button>` for creating buttons
  - `<p>` for text (paragraphs)
  - `<a>` for clickable links
  - `<a href="">` for assigning link for a text

## Exercises included
- **1a — Single button** (`1a-button.html`)
- **1b — Two buttons** (`1b-two-buttons.html`)
- **1c — Single paragraph** (`1c-Hello-World.html`)
- **1d — Two paragraphs** (`1d-two-paragraphs.html`)
- **1e — Text to link (non-redirect)** (`1e-text-to-link.html`)
- **1f — Redirecting link** (`1f-text-redirectory-link.html`)
- **1g — Final combined exercise** (`1g-final-exercise.html`)

## How to run
Open any `.html` file in a browser (double click the file), or right-click → **Open with** → browser.

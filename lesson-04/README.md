# Lesson 04 — CSS Spacing & Button Effects (Hover / Active)

This folder contains my completed exercises from **Lesson 04** of the *SuperSimpleDev HTML & CSS Full Course*.

## What I learned
- Creating spacing using `padding` and `margin`
- Using `:hover` and `:active` for button interaction
- Using `transition` to make effects smooth
- Using `box-shadow` to create a pressed button effect

## Exercises included
- **4h — Pagination (Back/Next + links with spacing)** (`4h.html`)
- **4i — Stretch button (padding change on hover)** (`4i.html`)
- **4j — 3D press effect (box-shadow + margin on active)** (`4j.html`)
- **4k — Stretch buttons (Fixed margin in between)** (`4k-final-exercise.html`)

## How to run
Open any `.html` file in a browser (double click the file), or right-click → **Open with** → browser.

# Lesson 02 — CSS Basics (Button Styling)

This folder contains my completed exercises from **Lesson 02** of the *SuperSimpleDev HTML & CSS Full Course*.

## What I learned
- Writing CSS inside a `<style>` tag
- Using CSS selectors:
  - Element selector (`button { ... }`)
  - Class selector (`.class-name { ... }`)
- Styling buttons using basic CSS properties:
  - `background-color`, `color`
  - `border`, `border-style`, `border-color`
  - `width`, `height`
  - `border-radius`
  - `font-size`, `font-weight`
  - `cursor`, spacing with `margin-right`
- Using `rgb()` color values
